## 📋 Project Description

This project detects fake news using Deep Learning. It was trained on a large dataset of real and fake news articles using TensorFlow/Keras and a CNN architecture for text classification.

**Accuracy achieved: 99.60%**

---

## ✨ Features

- Clean and intuitive web interface
- Real-time news classification
- High confidence scores
- Built with CNN + NLP techniques
- Responsive design
